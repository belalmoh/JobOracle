## UI Components Needed

### Resume Upload
- **Dropzone**: Large drag-and-drop area with file type icons
- **Progress Bar**: Linear progress indicator during upload/processing
- **Error Toast**: Destructive-colored toast for upload failures

### Parsed Data Display
- **Collapsible Sections**: Card with collapse/expand for Skills, Experience, Education
- **Toggle Switch**: View toggle between parsed data and source text
- **Tags/Chips**: For skill display with remove button

### Keyword Management
- **Input + Enter**: Text input with Enter key handler
- **Bulk Edit Modal**: Textarea for comma-separated keywords

### Settings Config
- **Setup Wizard**: Multi-step dialog/wizard flow
- **Checkbox Group**: For job source selection
- **Preview Screen**: Final preview before save
