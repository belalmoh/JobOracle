       - Calculate keyword overlap score (30% weight)
       - Calculate semantic similarity using Ollama (70% weight)
       - Update Job.compatibility_score and Job.match_keywords
       - Return: { score: float, matched_keywords: string[], explanation: string }

    Implementation of scoring:
    ```python
    # Keyword overlap (simple word matching)
    job_text = f"{job.title} {job.description or ''} {job.requirements or ''}"
    job_text_lower = job_text.lower()
    matched = [kw for kw in user_keywords if kw.lower() in job_text_lower]
    keyword_score = len(matched) / max(len(user_keywords), 1) * 100

    # Semantic similarity (Ollama)
    # Use same Ollama client as parse.py
    ollama_response = ollama.chat(
        model='nomic-embed-text',
        messages=[{
            'role': 'user',
            'content': f"Rate job match (0-100): Job: {job_text[:500]}... Resume keywords: {user_keywords}"
        }]
    )
    # Parse score from response or use embeddings comparison
    ```

    Match explanation format:
    "Your keywords '{matched_keywords}' match this job. Overall match: {score}%"
